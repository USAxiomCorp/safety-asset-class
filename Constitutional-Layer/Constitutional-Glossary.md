# Constitutional Layer (Tier‑0) — Glossary
## Foundation for Aligned Intelligence, Truth, and Humanity (FAITH)

This glossary defines the formal terminology used throughout the Constitutional Layer (Tier‑0).  
All definitions are normative and required for SAC certification, refinement safety, legal compliance, deterministic execution, immutable auditability, and hardware‑level enforcement.

These terms form the deepest substrate of aligned intelligence.

---

## A

### **Aligned State**
A system state that satisfies all invariants, constraints, and legal requirements.

### **Audit Replayability**
The requirement that all constitutional evaluations must be reproducible from RAC traces.

---

## B

### **Boundary Condition**
A mathematical or ethical limit that defines the edge of the Constitutional Boundary Set.

### **Boundary Distance**
A metric measuring proximity to the constitutional boundary.

### **Boundary Set (\(\mathcal{B}\))**
The set of all states permitted by the Constitution.

---

## C

### **Constitutional Constraint**
A rule that must hold for all system states:
\[
C(S) = \text{true}
\]

### **Constitutional Invariant**
A property that must remain true for all system states:
\[
I(S) = \text{true}
\]

### **Constitutional Metric**
A quantitative measure used to evaluate safety, legality, stability, or uncertainty.

### **Corrective Projection**
A projection applied when a state leaves the boundary set.

---

## D

### **Deterministic Evaluation**
The requirement that constitutional evaluations produce identical results for identical inputs.

### **Deterministic Projection**
A projection operator that produces the same output for the same input across all implementations.

---

## E

### **Escalation**
A mandatory halt
